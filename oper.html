Microsoft Windows

Windows — это операционная система, разработанная корпорацией Microsoft. Она предоставляет пользователю удобную и интуитивно понятную среду для работы с компьютером. Windows обеспечивает управление ресурсами компьютера, взаимодействие с аппаратным обеспечением, поддержку различных приложений и доступ к сетям.

Android Google

Android — это операционная система для различных гаджетов (смартфоны, планшеты, электронные книги, смарт-часы, фитнес-трекеры и т. д.). Относительно недавно её начали использовать даже в автомобилях и роботизированных бытовых комплексах.
